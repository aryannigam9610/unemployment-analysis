# 📊 Unemployment Analysis Using Python

An end-to-end Data Analysis project that explores, visualizes, and uncovers trends in the unemployment rate across different states and regions (Urban vs. Rural).

## 📌 Project Overview
Unemployment is a critical economic metric. This project performs Exploratory Data Analysis (EDA) on historical unemployment data to understand the sharp spikes, regional differences, and identify the hardest-hit areas using advanced data visualization techniques.

## ✨ Features
* **Regional Comparison:** Compares unemployment dynamics between Rural and Urban areas.
* **Geographical Insights:** Identifies the top states struggling with high average unemployment rates.
* **Outlier Detection:** Highlights extreme anomalies and economic spikes using box plots.
* **Data Processing:** Cleans and structures raw economic data using Pandas.

---

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Seaborn, Matplotlib
